Бэкэндом (back-end) называется часть транслятора, 
которая уже не работает с текстовым представлением программы. 
Вместо этого работа ведется с объектной моделью программы — графом объектов.

Объектную модель формируют экземпляры наследников класса Term.
Все необходимые для работы трансляторы операции над объектной моделью воплощены
в виде методов класса Term.

Бэкенд знает лишь о самых фундаментальных особенностях языка. 
Вся конкретика (типы данных, библиотечные функции, ...) 
должны определяться в пакете ru.usu.cs.fun.lang.
