На данный момент в проекте присутствуют либо планируются следующие пакеты:

ru.usu.cs.fun.utils
	библиотека со вспомогательными классами и утилитами, не завязанными на язык fun.
	допустимые зависимости от: —
	
ru.usu.cs.fun.engine 
	библиотека, реализующая основные базовые концепции языка
	допустимы зависимости от: utils.

ru.usu.cs.fun.lang
	библиотека с менее концептуальными вещами — типами данных, функциями, и т.д.
	допустимы зависимости от: utils, engine.

ru.usu.cs.fun.repl
	запускаемое приложение — REPL среда программирования для fun
	допустимы зависимости от: utils, engine, lang.

ru.usu.cs.fun.interpreter
	запускаемое приложение — консольный интерпретатор языка fun.
	допустимы зависимости от: utils, engine, lang.

ru.usu.cs.fun.formatter
	запускаемое приложение — утилита для переформатирования программ на языке fun.
	допустимы зависимости от: utils, engine, lang.

ru.usu.cs.fun.testing
	функциональные тесты на язык fun.
	допустимы зависимости от: любые.
