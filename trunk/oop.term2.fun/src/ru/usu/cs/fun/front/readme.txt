Фронтендом (front-end) называется часть транслятора, 
которая работает с текстовым представлением программы, переводя ее в объектную модель. 

Фронтенд состоит из обобщенного парсера и обобщенного лексера.
Лексер разбивает текст на лексемы, а парсер строит объектную модель из потока лексем.

Данный пакет предоставляет обобщенный механизм для создания фронтенда.
Вся конкретная специфика языка (типы лексем, грамматика, ...) 
должны определяться в пакете ru.usu.cs.fun.lang.
